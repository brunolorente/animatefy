Hover animations

Background Animations

Slide Right
Slide Left
Slide Out
Fade
Angle Out

Explanation:

When you are going to use some animation, please, keep in mind: An effect’s class name is a lower case version of its name, prefixed by afy-hover-, in this case .afy-hover-slideright (the dot represents a class and the prefix makes the class name a little more unique so it’s not already in use). Where a name has a space in it, such as Slide Right, then the space is replaced with nothing, please, erase it, like so: .afy-hover-slideright. Remember, you need to use just lowercase letters.

CSS Class:

.afy-hover-slideright

Sample code:

<div class="afy-hover-slideright">
 ...
</div>

Explanation:

When you are going to use some animation, please, keep in mind: An effect’s class name is a lower case version of its name, prefixed by afy-hover-, in this case .afy-hover-gradientfade (the dot represents a class and the prefix makes the class name a little more unique so it’s not already in use). Where a name has a space in it, such as Gradient Fade, then the space is replaced with nothing, please, erase it, like so: .afy-hover-gradientfade. Remember, you need to use just lowercase letters. What about gradient colors? Don't worry you can establish them using gradient-first-color=""and gradient-second-color="" atributes, so you can use hex format or something you would use inside your CSS.

CSS Example Class:

.afy-hover-gradientfade

HTML attribute:

gradient-first-color="aquamarine" gradient-second-color="dodgerblue"

Sample code:

<div class="afy-hover-gradientfade" gradient-first-color="aquamarine" gradient-second-color="dodgerblue">
 ...
</div>

Scroll animations

Technically, the Animatefy engine detects what elements are setted up with their classes like .animatefy-slide-left and genetates event listeners that detect the exactly moment that user sees for first time the element. Just in time, the animation is trigered and the user starts to love your website. Don't be afraid, you just need to follow the installation instructions and then, to emulate the instructions from the right side (or bellow if you are reading this from small screen :D)

Parallax

No worries, create a parallax effect never was so easy, the Animatefy engine makes the magic for you. You just need to follow the following steps:

  1. Install Animatefy inside your website
  2. Copy the sample code
  3. Enjoy your new full responsive parallax

Sample code:

<div class="parallax-container">
 <div class="animatefy-parallax">
  <p class="caption">
   Animatefy Parallax!
  </p>
 </div>
</div>
